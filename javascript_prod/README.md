# Выпускная работа по курсу Ярков А.В.

## Установка
* Установите [Node.js](https://nodejs.org/en/)
* Установите [npm](https://docs.npmjs.com/cli/v8/commands/npm-install)
`brew install npm`
* Установите зависимости проекта `npm install`

## Для разработки:
`npm run dev`

## Сборка:
`npm run build`
